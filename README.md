# 配置相关

> 基于ThinkPHP 6开发
> 运行环境要求PHP7.1+。

数据库及其他配置修改env文件即可，默认为

```
APP_DEBUG = false

[APP]
DEFAULT_TIMEZONE = Asia/Shanghai

[DATABASE]
TYPE = mysql
HOSTNAME = 127.0.0.1
DATABASE = mail
USERNAME = root
PASSWORD = root
HOSTPORT = 3306
CHARSET = utf8
DEBUG = false

[LANG]
default_lang = zh-cn
```

后台默认账号密码（密码修改直接替换库中MD5即可）

```
admin|123456
```
